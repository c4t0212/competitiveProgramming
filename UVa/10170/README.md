# [UVa 10170](https://vjudge.net/problem/UVA-10170)  

給定s和d, 求 s+(s+1)+(s+2)+(s+3)+...+(s+n)何時能超過d  

* sol-1:  
  暴力解, 用迴圈慢慢跑下去就好, 還好不會TLE  
 
* sol-2 & sol-3:  
  上面的數學式可以整理成s+(s+1)+(s+2)+(s+3)+...+(s+n) < d, 可以拆成兩部分  
  s + s + ... + s + 1 + 2 + 3 + ... + n < d  
  (n + 1) * s + n * (n + 1) / 2 < d 通分整理一下變成
  (n + 1)(2s + n) < 2d  
  兩種不同寫法的二分搜, 搜
